# Ημερολόγιο του φαρμακοποιού - Website

Website for the annual pharmacist calendar by Phasma Promotions.

## Files Structure

- `index.html` - Main HTML file
- `styles.css` - Stylesheet with pharmacy green color scheme
- `script.js` - JavaScript for smooth scrolling and navigation
- `images/` - Directory for calendar cover images (create this folder and add images)

## Setup Instructions

1. Create an `images` folder in the project directory
2. Add calendar cover images:
   - `2025-cover.jpg` - Current year cover
   - `2024-cover.jpg`, `2023-cover.jpg`, etc. - Archive covers
3. Update the phone number in `index.html` (search for `[Εισάγετε αριθμό τηλεφώνου]`)
4. Customize the content in the About section if needed

## Features

- Responsive design
- Smooth scrolling navigation
- Minimal, colorful design with pharmacy green theme
- Greek language content
- Archive section for past editions
- No purchase/signup functionality (phone orders only)
